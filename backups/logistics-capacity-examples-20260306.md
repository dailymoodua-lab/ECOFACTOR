# Logistics Per-Car Examples (2026-03-06)

## Формула
`(база контейнера + добавка до Одессы) / кол-во авто в контейнере`

## Для 3 авто (обычно)
- Клайпеда -> Одесса: `(4550 + 750) / 3 = 1766.67 USD/авто`
- Мостиська -> Одесса: `(6300 + 180) / 3 = 2160.00 USD/авто`
- Одесса -> Одесса: `(6800 + 0) / 3 = 2266.67 USD/авто`

## Для 4 авто (исключение Yuan Up)
- Клайпеда -> Одесса: `(4550 + 750) / 4 = 1325.00 USD/авто`
- Мостиська -> Одесса: `(6300 + 180) / 4 = 1620.00 USD/авто`
- Одесса -> Одесса: `(6800 + 0) / 4 = 1700.00 USD/авто`
