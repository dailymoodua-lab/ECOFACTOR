## Контейнер: базовая стоимость по точке входа
- Клайпеда: `$4,550` за контейнер
- Мостиська: `$6,300` за контейнер
- Одесса: `$6,800` за контейнер

## Доставка до Одессы (добавка)
- Из Клайпеды до Одессы: `+$750`
- Из Мостиська до Одессы: `+$180`
- Если точка входа Одесса: `+$0`

## Принцип учёта цены контейнера
- Цена контейнера учитывается как комплексная (единая стоимость по маршруту/точке входа).

## Вместимость контейнера
- Базово: `3 авто/контейнер`
- Исключение: `BYD Yuan Up` — `4 авто/контейнер`
- Правило для больших авто: использовать `3 авто/контейнер`

## Формулы (для будущих правок)
- `container_total_usd = base_container_usd + delivery_to_odessa_usd`
- `logistics_per_car_usd = container_total_usd / cars_per_container`
- Где `cars_per_container`:
  - `4` для `Yuan Up`
  - `3` для остальных (в т.ч. больших авто)
